Extract Boot.img Directly from Device Without Downloading Firmware

With your device booted to TWRP and connected to PC, type in the below command in the CMD window that is opened inside the platform-tools folder:

adb.exe shell dd if=/dev/block/bootdevice/by-name/boot of=/tmp/boot.img

To take a backup of this file onto your PC as well do this script:

adb.exe pull /tmp/boot.img stock_boot.img

To exit adb shell:

adb.exe shell exit

To reboot device:

adb.exe reboot
